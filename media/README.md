| File | Purpose |
| --- | --- |
| `tvc_gimbal_motion.svg` | GitHub-renderable animated schematic of the Rev A pitch/yaw gimbal motion. |
| `tvc_gimbal_animation.html` | Browser-viewable interactive motion animation with pitch/yaw angle readouts. |

The animation is a kinematic explanation, not a structural or controls simulation. It shows how the yaw frame carries the pitch/nozzle subsystem, while the pitch axis rotates the inner carrier. That architecture is physically important because it explains why the yaw-axis inertia is higher and why the pre-hardware model predicts slower yaw response for the same useful servo torque.
